About me

Hi! I'm David Guerreiro, a spanish web developer based in London. I'm passionate about the web so I have moved to the UK in order to improve my skills as a web developer.

As a freelance, I have always used the latest standards to develop websites in a professional way. I have spent most of my time working whit PHP, MySQL, HTML5, CSS3, Javascript (including libraries like jQuery) and sometimes with responsive design as well. I'm also developing a videogame as a side project.

I write code using Sublime Text 2. I keep my task organized using Redbooth and my code using git.

When I have to write code, I always keep it neat and tidy. This is something really important for me because is really annoying to fix a program when you can't understand the code. For now, I always develop web sites from scratch, without frameworks. In my opinion, this is the best way to learn.

I really love playing basketball in my spare time.

At the moment, I'm looking for a web developer position, so if you need someone, don't hesitate to contact me. You can either fire me a tweet or send me an e-mail.
